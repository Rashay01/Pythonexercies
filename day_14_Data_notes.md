# Normilization:
- reduce the risk of anomalies

anomalies:
- Update - doesnt update all info
- Delete - you remove info of data hat shouldnt be deleted 

1NF:
- No row order
- Dont mix data types
- Needs Primary Key
    - Unique
    - Not Null
    - only one column 
- No repeating groups

2NF:
- Non-Key attributes depend solely on primary key
    - Non-key attributes


3nf:
- No Transitive dependancies
